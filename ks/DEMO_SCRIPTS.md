# KS2 Demo Scripts

Run from repo root:

```text
:script ks/<name>.ks2.txt
```

If you `:cd ks` first, run without the `ks/` prefix.

## New additions

- `demo-dw-bass-tight.ks2.txt` - tight modern mono bass, moderate drive
- `demo-dw-bass-aggressive.ks2.txt` - heavier drive + sharper attack
- `demo-dw-bass-chug.ks2.txt` - percussive palm-mute style chug pattern
- `demo-dw-bass-triplet.ks2.txt` - triplet-grid rolling bassline
- `demo-dw-bass-octave-run.ks2.txt` - octave jumps with movement
- `demo-dw-bass-octave-run-alt.ks2.txt` - brighter/dirty A-B variant
- `demo-dw-bass-subpulse.ks2.txt` - sub-forward pulse/saw blend
- `demo-dw-poly-stabs.ks2.txt` - polyphonic stabs/chord punctuation
- `demo-dw-stereo-space.ks2.txt` - stereo pan spread + pan LFO + delay send showcase
